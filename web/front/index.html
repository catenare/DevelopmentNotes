    Setting for static page development

    • Using foundation from zurb

    Installation - Foundation for Sites

    1. Install foundation-cli - npm install --global foundation-cli
    2. Create new project - foundation new --framework sites --template zurb
      • Use foundation-zurb-template
    3. Install building block - foundation kits install news
    4. foundation watch
